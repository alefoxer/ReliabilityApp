# Итоговое состояние проекта

Документ описывает готовое состояние приложения, его основные возможности, поддерживаемые сценарии и ограничения текущей версии.

## Общая характеристика

`Надёжность технических средств` — настольное приложение на Python и PyQt6 для построения структурных схем надежности, генерации расчетных формул, выполнения численных расчетов и подготовки отчетных материалов.

Основные модули приложения:

- графический редактор структурных схем;
- калькулятор методик F1.1-F7.2;
- генератор формул и объяснений;
- экспорт отчетов и расчетных данных.

Основные способы запуска:

```bash
python -m app.main
python main.py
```

На Windows также доступны `run_app.ps1` и `run_app.bat`.

## Поддерживаемый пользовательский сценарий

Приложение позволяет собрать или загрузить структурную схему, задать параметры элементов, получить общую формулу системы, выполнить расчет и сохранить результаты.

В графическом редакторе доступны встроенные шаблоны последовательных, параллельных, комбинированных и резервированных схем. Пользователь может редактировать блоки и связи, сохранять схему в JSON, загружать ее обратно и экспортировать изображение схемы в PNG или SVG.

После построения схемы приложение формирует символическую формулу надежности и коэффициента готовности. Для расчетов используются параметры элементов, выбранная методика и структура графа. Результаты отображаются в виде таблицы, графика, формулы под графиком и итогового вывода по заданному порогу.

## Формулы и расчетные модели

Расчетное ядро покрывает методики F1.1-F7.2 и поддерживает:

- последовательные схемы;
- параллельные схемы;
- смешанные последовательно-параллельные композиции;
- резервирование 1+1 как рабочую модель прототипа;
- вложенные схемы через `SchemeModel`;
- выбор расчетного сценария по структуре схемы;
- предупреждения для неиспользуемых или неоднозначных элементов.

Генератор формул ориентирован на ациклические структурные схемы с одним входом и одним выходом. Для очень сложных композиций формула может строиться как приближенная структурная интерпретация с предупреждением.

## Экспорт и отчетные материалы

Приложение поддерживает экспорт:

- отчетов в HTML, TXT, PDF, DOCX и XLSX;
- расчетных таблиц и данных графика;
- изображений схемы в PNG и SVG;
- пользовательских схем в JSON.

HTML-отчет содержит титульный блок, методику, формулы, результаты, график, таблицы и заключение. DOCX-отчет ориентирован на передачу в текстовом редакторе. XLSX-файл содержит отдельные листы для сводки, исходных данных, результатов, формул, данных графика, графиков и заключения.

## Проверяемость проекта

В проект включен набор автоматических тестов для расчетного ядра, генератора формул, выбора методик, сериализации схем, валидации данных и экспорта отчетов.

Команда запуска тестов:

```bash
python -m pytest
```

Также в приложении есть встроенная самопроверка генератора формул, доступная из интерфейса.

## Ограничения текущей версии

- Генератор формул рассчитан на ациклические структурные схемы.
- Резервирование в графическом редакторе реализовано как рабочая модель прототипа.
- Для части композиционных формул источник обозначается как `Методическая формула проекта`, если отдельная ссылка на стандарт не задана в материалах.
- Сложные циклические графы и промышленная шаблонизация офисных отчетов не входят в возможности текущей версии.
