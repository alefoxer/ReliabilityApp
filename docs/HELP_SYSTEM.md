# Встроенная справка

Этот документ описывает, где лежат тексты справки и как их обновлять.

## Файлы

- `help_content.py` — встроенные тексты приложения.
- `docs/USER_GUIDE.md` — полное руководство пользователя.
- `gui_main.py` — пункты меню `Справка`, `О программе`, `Открыть руководство пользователя`.

## Как устроено окно справки

В `help_content.py` есть список `HELP_TOPICS`. Каждый элемент — это пара:

```python
("Название темы", "<h2>HTML-текст темы</h2>")
```

Окно справки в `MainWindow.show_help()` строит список тем слева и показывает выбранный HTML-текст справа через `QTextBrowser`.

## Как обновлять справку

1. Обновите короткую встроенную тему в `help_content.py`.
2. Обновите подробный раздел в `docs/USER_GUIDE.md`.
3. Если изменилась логика меню или пути к файлам, обновите `gui_main.py`.
4. Запустите `python -m compileall -q .` и `python -m pytest`.

## Руководство пользователя

Пункт `Открыть руководство пользователя` ищет `docs/USER_GUIDE.md` рядом с исходниками или рядом с исполняемым файлом собранного приложения. Если файл найден, он открывается через `QDesktopServices`. Если файл отсутствует или системный просмотрщик не смог его открыть, пользователь получает понятное предупреждение.
